# تعليمات النشر

هذا README يوضح كيفية نشر مشروع "بنت اليمن".

## المتطلبات
1. Git
2. Node.js (الإصدار 14 أو أعلى)
3. npm (الإصدار 6 أو أعلى)
4. قاعدة بيانات (على سبيل المثال، MySQL)

## خطوات النشر

### 1. استنساخ المستودع

قم باستنساخ المستودع إلى جهازك:
```bash
git clone https://github.com/G24-maker/bint-alyemen-storee.git
cd bint-alyemen-storee
```

### 2. تثبيت الاعتماديات

بعد الاستنساخ، ثبّت الاعتماديات:
```bash
npm install
```

### 3. إعداد قاعدة البيانات

قم بإنشاء قاعدة بيانات جديدة وقم بتحديث ملف الإعدادات بالاتصال بقاعدة البيانات.

### 4. تشغيل المشروع

ابدأ تشغيل المشروع باستخدام:
```bash
npm start
```

### 5. فتح المتصفح

افتح المتصفح وانتقل إلى:
```
http://localhost:3000
```
